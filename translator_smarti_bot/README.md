# Translator Smart Bot

AI-powered translation workflow built with :contentReference[oaicite:0]{index=0} and OpenAI API.

## Overview

Translator Smart Bot automatically translates text into any language.  
The user only needs to provide:

- input text
- target language

The workflow validates the input data before processing and returns the translated text.

## Features

- Text translation into multiple languages
- Input validation before translation
- AI-powered language processing
- Fast automated responses
- Simple and user-friendly workflow

## Workflow Logic

1. User sends text and target language
2. Workflow validates input data
3. AI processes translation request
4. Translated text is returned to the user

## Tech Stack

- n8n
- OpenAI API
- AI Automation
- Telegram
